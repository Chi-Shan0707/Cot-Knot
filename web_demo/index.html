💡 Why This Matters

🔬 Measurement Theory

Demonstrates measurement invariance failure: same features measure different constructs across domains.

🚫 Practical Limits

Shows where cheap verifiers work (math) and where they fail (coding)—guides tool development.

🎯 Domain Awareness

Highlights need for domain-specific verification strategies, not one-size-fits-all approaches.

⚡ Execution Matters

For coding, execution-aware verification is likely essential—text features are fundamentally limited.

🎯 Real-World Impact: Best-of-N Reranking

Math: +10.0% pass@1
Science: +8.0% pass@1
Coding: -0.6% (no improvement)

📝 What We Do & Don't Claim

✓ We Show

  • Hand-crafted CoT-surface features work well in math
  • They have narrow utility in science
  • They fail for coding correctness prediction
  • This is a measurement failure, not modeling failure

✗ We Don't Claim

  • All text-based verifiers must fail on coding
  • Execution-aware or code-aware methods won't work
  • LLM judges or raw-text classifiers are useless
  • Hidden-state probes can't capture correctness

⚙️ Technical Details

Model: DeepSeek-R1-0528-Qwen3-8B

Science Data: GPQA (12,672 runs)
